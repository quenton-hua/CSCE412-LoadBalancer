Detailed Description

Class representing a web server that processes requests.

The WebServer class handles requests assigned by the load balancer. It tracks whether the server is busy, how much time is left to complete the current request, and when the request started.

Simulate the passage of time by decrementing the time left to process the request.

This method simulates the progression of time in the system by reducing the remaining time for the current request. If the request completes, the server becomes available for new requests.
