\doxysubsection{Detailed Description}
Class representing a web server that processes requests. 

The \mbox{\hyperlink{classWebServer}{Web\+Server}} class handles requests assigned by the load balancer. It tracks whether the server is busy, how much time is left to complete the current request, and when the request started.
